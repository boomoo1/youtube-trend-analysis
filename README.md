# 📊 YouTube Trend Analysis
> 마냥 성실한 시대는 지났다. 성공을 위한 유튜브 공략법

## 1. 프로젝트 개요
유튜브 트렌딩 영상 데이터를 기반으로
조회수에 영향을 주는 요인들을 분석하였다.

분석 질문:
1. 카테고리에 따라 조회수 차이가 존재하는가?
2. 영상 제목 길이는 성과에 영향을 주는가?
3. 태그 개수는 조회수에 어떤 영향을 미치는가?
4. 업로드 요일과 시간대는 조회수와 관련이 있는가?

## 2. 사용 기술
- Python (pandas, matplotlib)
- Tableau
- GitHub

## 3. 데이터
- 출처: Kaggle - YouTube Trending Dataset
- 약 39만 개 영상

## 4. 주요 인사이트
- 카테고리별 조회수는 집계 방식(SUM/AVG/MEDIAN)에 따라 다르게 해석됨
- 평균 조회수는 소수의 바이럴 영상에 의해 왜곡될 수 있음
- 제목 길이·태그 수는 조회수와 약한 상관관계를 보임
- 업로드 요일 × 시간대 조합이 조회수에 영향을 미침
