# 📊 YouTube Category View Analysis

## 1. 프로젝트 개요 (Overview)

본 프로젝트는 **YouTube Trending Dataset**을 활용하여  
유튜브 **카테고리별 조회수 차이**를 다양한 집계 방식(SUM, AVG, MEDIAN)으로 분석하고,  
집계 기준에 따라 해석이 어떻게 달라지는지를 확인하는 것을 목표로 합니다.

단순 평균 조회수 비교의 한계를 인식하고,  
시장 규모·대표성·바이럴 영향 등을 함께 고려한 데이터 기반 인사이트를 도출합니다.

---

## 2. 분석 질문 (Research Questions)

- 카테고리별 전체 조회수 규모는 어떻게 다른가?
- 평균 조회수가 높은 카테고리는 어디인가?
- 중앙값을 기준으로 보면 결과는 어떻게 달라지는가?
- 왜 집계 방식에 따라 카테고리 해석이 달라지는가?

---

## 3. 데이터 설명 (Dataset)

- **출처**: Kaggle – YouTube Trending Dataset
- **데이터 특징**:
  - YouTube 트렌딩 영상 기준 데이터
  - 다수 국가의 트렌딩 영상 포함
- **주요 컬럼**
  - `video_id`
  - `title`
  - `categoryId`
  - `view_count`
  - `likes`
  - `comment_count`
  - `publishedAt`

---

## 4. 데이터 전처리 (Data Preprocessing)

- 분석에 필요한 컬럼만 선택
- 조회수(`view_count`) 기준 분석
- 카테고리별 영상 단위 집계
- 결측치 및 이상치는 집계 단계에서 자연스럽게 반영되도록 처리
- 바이럴 영상으로 인한 왜곡을 고려하여 **중앙값(MEDIAN)** 함께 사용

---

## 5. 분석 방법 (Methodology)

### 5.1 집계 지표 정의

| 지표 | 설명 |
| --- | --- |
| SUM(View Count) | 카테고리 전체 조회수 합 |
| AVG(View Count) | 영상 1개당 평균 조회수 |
| MEDIAN(View Count) | 대표적인(전형적인) 조회수 |

### 5.2 집계 방식을 나눠서 분석한 이유

- **SUM**: 시장 규모 및 전체 영향력 파악
- **AVG**: 카테고리 경쟁력 비교
- **MEDIAN**: 소수의 바이럴 영상으로 인한 평균 왜곡 제거

---

## 6. 주요 분석 결과 (Key Findings)

### 6.1 SUM 기준 분석
- Music, Entertainment, Gaming 카테고리가 전체 조회수에서 큰 비중 차지
- 업로드 수와 대중성이 높은 카테고리가 시장 규모를 주도

### 6.2 AVG 기준 분석
- Pets & Animals, Entertainment, Music 카테고리가 높은 평균 조회수
- 영상 수가 적은 카테고리는 소수의 인기 영상으로 평균이 상승하는 구조 확인

### 6.3 MEDIAN 기준 분석
- 평균 대비 중앙값이 낮은 카테고리는 바이럴 의존도가 높음
- Music 카테고리는 평균과 중앙값의 차이가 비교적 작아 안정적인 조회수 구조를 보임

---

## 7. 인사이트 & 해석 (Insights)

- **시장 규모(SUM)** 와 **콘텐츠 경쟁력(AVG)** 은 반드시 일치하지 않음
- 평균 조회수만으로 카테고리를 평가할 경우 왜곡 가능성 존재
- 중앙값 분석을 통해 “일반적인 영상 성과”를 더 정확히 파악 가능
- 콘텐츠 기획 및 마케팅 전략 수립 시 집계 지표의 선택이 매우 중요함
